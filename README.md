# dotfiles
My dotfiles.

# keybind
```
sudo apt update
sudo apt install xmodmap
```

# vim
```
sudo apt update
sudo apt install vim-gtk3
```
Then install Neobundle to install vim plugins.

# fzf
https://github.com/junegunn/fzf
```
git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install
```
